Moto G 2014 ToolKit luca020400

This tool include :

1) Universal Moto G 2014 Support

2) Unlock / Relock the bootloader

3) TWRP and PHILZ TOUCH ( root with a recovery ) ( You can flash or boot it )

4) Automatic Root Script

5) Remove/Restore Unlocked Bootloader Warning

How to use :

Drivers :

Windows : https://www.motorola.com/getmdmwin

Mac OS X : https://www.motorola.com/getmdmmac

Linux : Already Included

Reboot your phone into bootloader ( Volume Down and Power Button )

Linux and Mac OS X:

./bootloader.sh unlock/relock
./root.sh
./twrp.sh boot/flash
./philz.bat boot/flash
./logo.sh nowarning/warning

Windows :

bootloader.bat unlock/relock
root.bat
twrp.bat boot/flash
philz.bat boot/flash
logo.bat nowarning/warning
