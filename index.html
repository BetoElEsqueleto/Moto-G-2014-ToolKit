Moto G 2014 ToolKit
download .ZIPdownload .TGZ

Tool for the Moto G 2014

This tool include :

1) Unlock / Relock the bootloader

2) TWRP and PHILZ TOUCH ( root with a recovery ) ( You can flash or boot it )

3) @Chainfire Autoroot ( root without a recovery )

4) Remove/Restore Unlocked Bootloader Warning

How to use :

Reboot your phone into bootloader ( Volume Down and Power Button )

Linux and Mac OS X:

./bootloader.sh unlock/relock
./root.sh titan_$device ( like titan_retaildsds )
./twrp.sh boot/flash
./philz.bat boot/flash
./logo.sh nowarning/warning

Windows :

bootloader.bat unlock/relock
root.bat titan_%device ( like titan_retaildsds )
twrp.bat boot/flash
philz.bat boot/flash
logo.sh nowarning/warning
